## 3. 电源方案

### 3.1 电源需求分析

| 工作模式 | 电流估算 | 持续时间/频率 | 日耗电(mAh) |
|----------|----------|---------------|-------------|
| 深度睡眠 | 50μA | 23.5h/天 | 1.18 |
| BLE 广播 | 5mA | 按需，平均 0.1h/天 | 0.5 |
| 指纹采集+匹配 | 80mA | 10次/天，每次 1.5s | 0.33 |
| 电机驱动(开锁) | 500mA | 10次/天，每次 0.5s | 0.69 |
| WiFi 连接 | 120mA | 按需，平均 0.3h/天 | 36 |
| **合计** | — | — | **≈ 38.7** |

### 3.2 电池选型

| 参数 | 值 | 说明 |
|------|-----|------|
| 电池类型 | [8节 AA 碱性 / 5000mAh 锂电] | — |
| 总容量 | [12000mWh@1.5V / 18500mWh@3.7V] | — |
| 可用容量(80%) | [9600mWh] | 留 20% 安全余量 |
| 目标续航 | 12个月 | — |
| 实际可用天数 | 9600/(38.7mAh×3.7V) ≈ X 天 | 需与日耗电模型对齐 |

### 3.3 电源树

```
电池 → 保护电路 → 升压/降压
           ├── 3.3V LDO → MCU + 传感器
           ├── 5V Boost → 电机驱动
           └── 1.8V LDO → [可选] 摄像头
```

## 4. 射频设计

### 4.1 天线要求

| 参数 | 要求 | 说明 |
|------|------|------|
| 天线类型 | [板载 PCB 天线 / 陶瓷天线 / 外置棒状] | 根据结构决定 |
| 频率范围 | 2.4-2.4835GHz | WiFi/BLE/Zigbee |
| 增益 | ≥ 2dBi | — |
| 驻波比(VSWR) | ≤ 2.0 | 中心频率处 |
| 净空区 | ≥ 5mm（PCB天线）/ 无要求（陶瓷） | 结构设计必须保证 |

### 4.2 射频匹配

- π 型匹配网络预留（串联 0Ω + 并联 NC + 并联 NC）
- 推荐预留 U.FL 测试座用于传导测试
- 量产前需完成天线匹配调试
